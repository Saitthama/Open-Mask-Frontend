# Open-Mask

## Open-Source Augmented-Reality App für kreative Fotofilter

## „Filter ohne Grenzen, Kreativität ohne Limits!“



### Kurzbeschreibung (Nutzen):

Filter sind digitale Effekte, die per Bildverarbeitung Aufnahmen verändern. Diese gibt es heutzutage als Portraitfilter hauptsächlich auf Social-Media-Plattformen. Dort sind sie normalerweise nicht Open-Source und nicht mit anderen Plattformen kompatibel. Ohne Zusatzsoftware können sie weder erstellt noch bearbeitet werden. Ziel ist eine Open-Source App, mit der visuelle Filter erstellt, bearbeitet, angewandt und geteilt werden können.
